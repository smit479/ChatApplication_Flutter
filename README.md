# 📱 ChatApplication_Flutter

A Flutter-based chat application project.

---

## 🚀 Getting Started

Follow the steps below to download and run the project locally.

---

## 📥 Download Project (ZIP)

1. Go to the GitHub repository
2. Click on **Code → Download ZIP**
3. Extract the ZIP file on your system

---

## 🛠️ Requirements

Make sure you have the following installed:

* Flutter SDK **3.38.5**
* Dart SDK (comes with Flutter)
* Android Studio or VS Code
* Emulator or physical device

---

## 🔧 Setup & Run

### 1. Open Project

Open the extracted folder in:

* Android Studio
  OR
* VS Code

---

### 2. Check Flutter Version

Run:

```
flutter --version
```

Make sure it shows:

```
Flutter 3.38.5
```

---

### 3. Get Dependencies

Run:

```
flutter pub get
```

---

### 4. Run the App

Connect a device or start emulator, then run:

```
flutter run
```

---

## 📦 Build APK

To generate APK:

```
flutter build apk
```

APK location:

```
build/app/outputs/flutter-apk/app-release.apk
```

---

## ❗ Common Issues

### 🔹 Gradle Error

Run:

```
flutter clean
flutter pub get
```

---

### 🔹 SDK Version Issue

Make sure Flutter version is **3.38.5**

---

### 🔹 Device Not Detected

Run:

```
flutter devices
```
